# T-Ecosystem

T-Ecosystem is a unified project containing various components and applications for the T-EK ecosystem.

## Project Structure

This project contains the following repositories:

- **[tek-ui](file:///c:/Users/turha/Desktop/Dev/Bireysel/Projeler/T-EK/T-Ecosystem/tek-ui)**: Core UI component library.
- **[Renderci](file:///c:/Users/turha/Desktop/Dev/Bireysel/Projeler/T-EK/T-Ecosystem/Renderci)**: Rendering engine and associated tools.
- **[t-Market](file:///c:/Users/turha/Desktop/Dev/Bireysel/Projeler/T-EK/T-Ecosystem/t-Market)**: Marketplace platform.
- **[T-SA](file:///c:/Users/turha/Desktop/Dev/Bireysel/Projeler/T-EK/T-Ecosystem/T-SA)**: Specification Analysis and tools.
- **[ENV-I-main](file:///c:/Users/turha/Desktop/Dev/Bireysel/Projeler/T-EK/T-Ecosystem/ENV-I-main)**: Environment Intelligence module.
- **[UPH-main](file:///c:/Users/turha/Desktop/Dev/Bireysel/Projeler/T-EK/T-Ecosystem/UPH-main)**: Unified Project Hub.
- **[Weave-main](file:///c:/Users/turha/Desktop/Dev/Bireysel/Projeler/T-EK/T-Ecosystem/Weave-main)**: Design and canvas application.

## Getting Started

### Prerequisites

- Node.js
- pnpm

### Installation

Dependencies have been installed for the primary projects:

```bash
# tek-ui
cd tek-ui && pnpm install

# Renderci
cd Renderci/code && pnpm install

# t-Market
cd t-Market && pnpm install

# T-SA
cd T-SA/code && pnpm install
```

## Development

Each project can be run individually from its respective directory.
